This software is for the Qutech Maxtrix DC wiretester hardware
This software is offered "As Is".

Version V1.4 and V1.7 are suitable for the first hardware edition of the DC wire tester and will not funtion properly on the latest version hardware.

Version V2.0 is only suitable for the latest hardware version.
This version has the functionality to "calibrate" and "compensate""  for offset voltages in the internal measuring amplifiers.
This willl offer better accuracy in the very low (les than 1 K Ohm) and very High (as is > 10M ohm) resistance values.
